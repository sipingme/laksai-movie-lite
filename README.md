# Laksai Movie Lite

一个简洁、高效的在线影视搜索与播放平台。

## 项目简介

Laksai Movie Lite 是一个纯前端实现的影视资源聚合搜索工具，无需后端服务器，直接通过浏览器访问即可使用。项目采用原生 HTML/CSS/JavaScript 开发，无框架依赖，代码简洁易维护。

## 主要特点

- **多源聚合**：内置 8 个优质视频资源源，覆盖全网影视内容
- **智能解析**：支持多集剧集自动识别，一键选择集数
- **响应式设计**：完美适配手机、平板、电脑等各种设备
- **暗黑科技风**：深色主题搭配薄荷绿强调色，视觉体验出色
- **极速加载**：优化的网络请求策略，多代理自动切换
- **无广告干扰**：纯净观影体验，无弹窗无植入

## 使用方法

### 在线访问

直接访问 GitHub Pages 部署地址：
```
https://你的用户名.github.io/Laksai Movie Lite/
```

### 本地使用

1. 克隆仓库到本地：
```bash
git clone https://github.com/你的用户名/Laksai Movie Lite.git
cd Laksai Movie Lite
```

2. 用浏览器打开 `index.html` 文件即可使用

### 搜索影片

1. 在搜索框输入影片名称
2. 点击/回车开始搜索
3. 等待搜索结果展示
4. 点击影片卡片进入播放

### 切换视频源

- 页面顶部提供 8 个视频源按钮
- 点击不同按钮切换资源源
- 如果某个源搜索不到内容，尝试切换其他源

### 播放多集影片

- 对于电视剧、综艺等多集内容
- 系统会自动展示集数选择界面
- 点击对应集数即可播放
- 播放中点击返回可回到集数选择

### 播放器操作

- **返回按钮**：退出当前播放
- **ESC 键**：快速关闭播放器
- **线路切换**：对于第三方解析链接，可切换不同播放线路
- **手机端**：支持全屏播放，手势控制

## 项目结构

```
Laksai Movie Lite/
├── index.html          # 主页面（单文件应用）
└── README.md           # 项目说明文档
```

项目采用单文件架构，所有代码（HTML/CSS/JavaScript）都在 `index.html` 中，便于部署和维护。

## 代码规范

本项目所有变量、函数、DOM 元素 ID 均使用 `laksai_` 前缀命名，避免全局命名空间污染：

- 变量：`laksai_currentSource`, `laksai_sources`
- 函数：`laksai_searchVideos()`, `laksai_playVideo()`
- DOM ID：`laksai_searchInput`, `laksai_videoList`
